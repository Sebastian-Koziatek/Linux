# 🐧 Linux - Kompleksowe Materiały Szkoleniowe

Repozytorium zawiera pełne materiały szkoleniowe do nauki systemu Linux - od podstaw po zaawansowane zagadnienia administracji systemem.

## 📚 Struktura Materiałów

### Dział 1 - Wprowadzenie do Systemu
- **I. Wprowadzenie do systemu Linux** - podstawowe pojęcia i filozofia systemów Linux/Unix
- **II. Podstawowa instalacja systemu Linux** - proces instalacji i konfiguracja początkowa
- **III. Środowiska graficzne** - przegląd i konfiguracja środowisk desktop (GNOME, KDE, etc.)
- **IV. SSH** - bezpieczne zdalne połączenia i zarządzanie serwerami
- **V. Managery pakietów** - instalacja oprogramowania (apt, yum, dnf)
- **VI. Korzystanie z pomocy w systemie** - man pages, info, dokumentacja

### Dział 2 - Podstawy Pracy z Systemem Linux
- **I. Edytory tekstu** - vi/vim, nano i inne edytory terminalowe
- **II. Praca z powłoką Bash** - skrypty, zmienne, automatyzacja zadań
- **III. Uprawnienia i konta administracyjne** - zarządzanie użytkownikami, sudo, chmod
- **IV. GRUB** - bootloader i zarządzanie procesem uruchamiania
- **V. Startowanie systemu** - systemd, init, zarządzanie usługami
- **VI. Deamony** - usługi systemowe działające w tle

### Dział 3 - Zarządzanie Systemem Linux
- **1. Zarządzanie dyskami** - partycjonowanie, formatowanie, montowanie
- **2. RAID** - redundancja i wydajność macierzy dyskowych
- **3. LVM** - Logical Volume Manager - elastyczne zarządzanie przestrzenią
- **4. System plików** - typy systemów plików (ext4, xfs, btrfs)
- **5. Konfiguracja systemu** - pliki konfiguracyjne i ich lokalizacje
- **6. Przeszukiwanie systemu** - find, grep, locate
- **7. Konfiguracja sieci** - interfejsy sieciowe, routing, DNS
- **8. Crontab** - harmonogram zadań i automatyzacja
- **9. IPTABLES** - firewall i bezpieczeństwo sieciowe
- **10. SELINUX** - Security-Enhanced Linux
- **11. Samba** - udostępnianie zasobów w sieci

### Dział 4 - Tematy Dodatkowe
- Logi systemowe i ich kolorowanie
- Screen - zarządzanie sesjami terminalowymi
- I inne zaawansowane tematy

## 📁 Dodatkowe Zasoby

- **grafiki** - katalog techniczny zawierający grafiki (proszę zignorować)
